# 📊 Implementação de Gráficos com Chart.js

Este projeto é uma página web simples que demonstra a utilização da biblioteca **Chart.js** para criação de gráficos.

Há dois gráficos:

* 📈 Um gráfico de linha com dados de temperatura e umidade ao longo do dia
* 📊 Um gráfico de barras com médias mensais de temperatura e umidade


## 🚀 Tecnologias utilizadas

* HTML
* JavaScript
* Chart.js


## 📉 Funcionalidades

* Exibição de dados de temperatura e umidade em tempo real (mockados)
* Comparação entre temperatura e umidade
* Visualização mensal de médias climáticas


## 📂 Estrutura

O projeto é composto por um único arquivo HTML contendo:

* Estrutura da página
* Dois elementos `<canvas>` para os gráficos
* Script com a configuração dos gráficos usando Chart.js

## 💡 Observação

Este projeto é apenas para fins de estudo e demonstração do uso do Chart.js (SP Tech).
